# Pokédex (HTML/CSS/JS)


Projeto simples de Pokédex que consome a API pública PokeAPI e exibe até 1025 Pokémon.

Como usar

1. Abra `index.html` no seu navegador (duplo clique ou arraste para a janela).
2. Pesquise por nome ou número no campo de busca.
3. Clique em um card para ver detalhes (os detalhes são carregados sob demanda).

Observações

- Não requer build; apenas abra `index.html`.
- Requer internet para acessar a PokeAPI e as imagens.
